<!-- GETTING STARTED -->
## Getting Started

In order to run this app, follow these criterias.

### Prerequisites

* Python

### Installation

1. Clone the repo
    ```
    git clone https://github.com/SAGAR-TAMANG/medic-ai-lyzr.git
    ```
2. Install requirements.txt
    ```
    pip install requirements.txt
    ```
3. Start the server
    ```
    python manage.py runserver
    ```
